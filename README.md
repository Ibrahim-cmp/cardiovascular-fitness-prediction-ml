Cardiovascular Fitness Prediction Using Classical ML

Overview

This project builds an interpretable machine learning system to predict cardiovascular disease using structured health data (70,000 records).

Methods Used

- K-Nearest Neighbors (KNN)

- Decision Trees

- Naïve Bayes

- Mutual Information Feature Selection

- PCA Dimensionality Reduction

Evaluation Metrics

- Accuracy

- Precision / Recall
  
- F1-Score

- ROC Curve & AUC

Best Model

- Decision Tree achieved:

- Accuracy: 72%

- AUC: 0.78

Tech Stack

- Python, Scikit-Learn, Pandas, NumPy, Matplotlib
